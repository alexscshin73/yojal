## S-06. 채팅/학습 화면 (핵심)

```
┌─────────────────────────────────┐
│ ← 뒤로    [🟠 인사말]    ? 힌트 │
├─────────────────────────────────┤
│                                 │
│  🦜                             │
│  ┌──────────────────────────┐   │
│  │ ¡Buenos días!            │   │  ← AI 메시지 (Teal 배경)
│  │ 좋은 아침이에요 🌅        │   │    (한국어 힌트, 작게)
│  └──────────────────────────┘   │
│  오전 6:00                      │
│                                 │
│           ┌──────────────────┐  │
│           │ Buenos días!     │  │  ← 내 메시지 (연회색)
│           └──────────────────┘  │
│                    오전 6:01    │
│                                 │
│  🦜                             │
│  ┌──────────────────────────┐   │
│  │ ⚠️ 거의 다 왔어요!        │   │
│  │ ─────────────────────── │   │
│  │ 내 답: "Buenos dias"    │   │
│  │ 정답:  "Buenos días" ✓  │   │  ← 교정 (초록)
│  │ ─────────────────────── │   │
│  │ 💡 악센트(í)를 기억해요  │   │  ← 설명
│  └──────────────────────────┘   │
│                                 │
│  🦜                             │
│  ┌──────────────────────────┐   │
│  │ 오늘 기분을 말해볼까요?   │   │
│  └──────────────────────────┘   │
│                                 │
├─────────────────────────────────┤
│ [스페인어로 답장해보세요...]  ➤  │
└─────────────────────────────────┘
```

**힌트 패널** (? 버튼 누르면 아래서 올라옴):
```
┌─────────────────────────────────┐
│ 💡 힌트                    ✕   │
│ ─────────────────────────────── │
│ • bien      → 좋다              │
│ • cansado/a → 피곤하다          │
│ • feliz     → 행복하다          │
│ • triste    → 슬프다            │
│                                 │
│ 예시: "Estoy bien, gracias"     │
└─────────────────────────────────┘
```

**단어 학습 모드** (상황 단어 타입일 때):
```
┌──────────────────────────────────┐
│ 오늘의 운전 단어 (2/5)  ████░░  │
│ ────────────────────────────── │
│                                  │
│          🚗 conducir             │
│          운전하다                │
│                                  │
│  [다음 단어 →]                   │
└──────────────────────────────────┘
```

## 화면 이동 흐름

```
[앱 최초 실행]
S-01 환영 → S-02 레벨 → S-03 루틴 → S-04 알림 → S-05 홈

[일반 사용]
푸시 알림 클릭 → S-06 채팅
S-05 홈 → S-06 채팅
S-05 홈 하단탭 → S-08 기록
S-05 홈 하단탭 → S-09 설정
S-09 설정 → S-07 루틴 설정

[채팅 종료]
S-06 채팅 ← 뒤로 → S-05 홈
```
