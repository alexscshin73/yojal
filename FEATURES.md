## 개략적 Phase 계획 (상세는 설계 후 확정)

### Phase 1 — MVP
- Ollama 설치 + 모델 로드
- FastAPI 백엔드 기본 구조
- Expo 앱 기본 구조
- 채팅 화면 + AI 대화 (텍스트)
- Cloudflare Tunnel 연결
- 푸시 알림 (시간 기반)

### Phase 2 — 루틴
- 루틴 설정 화면
- 학습 타입별 AI 프롬프트
- 학습 이력 저장
- 홈 화면 + 스트릭

### Phase 3 — 완성도
- 단어장 / 오답노트
- TTS (AI 목소리)
- STT (사용자 음성)
- 앱스토어 출시
