# FEATURES — 기능 로드맵 + 완료 기준

> 상태: 개발 게이트 진입 대기 중 (완료 기준 정의 완료, 승인 필요)
> 승인 여부: ⏳ 미승인

## 개발 게이트 — MVP 완료 기준

### D-01. Ollama 설치 + 모델 로드
**상태**: 대기 중
**완료 기준**:
- Ollama가 이 Linux 머신에 설치되어 실행 중
- 스페인어 학습에 적합한 모델 1개 선택 후 로드 완료
- 터미널에서 스페인어 질문 → 스페인어 응답 확인

---

### D-02. FastAPI 백엔드 기본 구조
**상태**: 대기 중
**완료 기준**:
- FastAPI 서버 실행 시 `/health` 접속 → `{"status": "ok"}` 응답
- `/chat` 엔드포인트에 메시지 전송 → Ollama 거쳐서 스페인어 응답 반환
- 학습 타입 6가지 각각 다른 AI 지시문(시스템 프롬프트) 구조 존재

**학습 타입 6가지**: 인사말 / 상황 단어 / 새 학습 / 복습 / 오답 복습 / 일기 쓰기

---

### D-03. Expo 앱 기본 구조
**상태**: 대기 중
**완료 기준**:
- 실제 폰 또는 시뮬레이터에서 Expo 앱 실행
- 하단 탭 3개 (홈 / 기록 / 설정) 전환 가능
- Yojal 브랜드 컬러 (#00897B) 적용된 기본 레이아웃

---

### D-04. 채팅 화면 + AI 대화
**상태**: 대기 중
**완료 기준**:
- DESIGN.md S-06 레이아웃대로 채팅 화면 표시
- 화면 진입 시 AI가 스페인어로 먼저 인사
- 내가 입력한 텍스트 → FastAPI → Ollama → 교정 포함한 응답 화면에 표시
- 힌트 버튼 누르면 힌트 패널 표시

---

### D-05. Cloudflare Tunnel 연결
**상태**: 대기 중
**완료 기준**:
- 집 밖 모바일 데이터 환경에서 앱 실행
- AI와 채팅이 정상 작동 (집 내부 Wi-Fi 없이)
- bunny-app의 Cloudflare Tunnel 설정 재활용

---

### D-06. 푸시 알림
**상태**: 대기 중
**완료 기준**:
- 루틴에 설정된 시간에 실제 폰으로 알림 수신
- 알림 문구에 학습 타입 표시 (예: "🌅 인사말 시간이에요!")
- 알림 탭하면 채팅 화면으로 바로 이동

---

## 개발 순서

```
D-01 → D-02 → D-03 → D-04 → D-05 → D-06
Ollama  백엔드  앱뼈대  핵심기능  외부접속  알림
```

D-04가 핵심. D-01~D-03이 기반. D-05, D-06은 D-04 완료 후 진행.
