# FEATURES — 기능 로드맵

> 각 기능은 완료 기준이 명확해야 시작할 수 있다.
> 한 번에 하나씩, 순서대로 진행한다.

## Phase 1: 뼈대 (지금 여기)

### F-01. 프로젝트 초기 설정
**완료 기준**: `npm run dev`로 브라우저에서 기본 화면이 뜬다
**상태**: 대기 중

### F-02. 메인 화면 UI
**완료 기준**: 브라우저에서 Yojal 로고, 언어 선택(스페인어/영어) 버튼이 보인다
**상태**: 대기 중

### F-03. 대화 화면 레이아웃
**완료 기준**: 채팅 UI (메시지 목록 + 입력창)가 화면에 표시된다
**상태**: 대기 중

### F-04. Claude API 연결
**완료 기준**: 채팅창에 한국어로 질문하면 AI가 스페인어/영어로 답한다
**상태**: 대기 중

### F-05. Vercel 배포
**완료 기준**: 인터넷 주소(URL)로 앱에 접속할 수 있다
**상태**: 대기 중

---

## Phase 2: 학습 기능

### F-06. 문법 교정 모드
**완료 기준**: 틀린 문장 입력 시 교정 + 설명이 표시된다
**완료 기준 추가**: 교정 내용이 원문과 구분되어 보인다
**상태**: 대기 중

### F-07. 레벨 설정
**완료 기준**: 초급/중급/고급 선택 시 AI 답변 난이도가 달라진다
**상태**: 대기 중

### F-08. 단어 퀴즈
**완료 기준**: 단어 5개를 보여주고 맞히면 점수가 올라간다
**상태**: 대기 중

---

## Phase 3: 고급 기능 (추후 결정)

- 발음 교정 (음성 입력)
- 학습 이력 저장
- 앱스토어 출시

---

## 기능 추가 규칙

새 기능을 추가하려면:
1. 이름, 완료 기준, Phase를 먼저 적는다
2. 순서(우선순위)를 결정한다
3. 현재 진행 중인 기능이 완료된 후 시작한다
