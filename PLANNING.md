## 3. 핵심 컨셉: 하루 루틴 기반 학습

사용자가 1주일 루틴을 설정하면, AI가 그 시간에 맞춰 먼저 말을 건다.

**예시 루틴**:
| 시간 | 요일 | 학습 타입 | 예시 |
|------|------|-----------|------|
| 06:00 | 매일 | 인사말 | ¡Buenos días! ¿Cómo dormiste? |
| 08:00 | 평일 | 상황 단어 | 운전/교통 관련 단어 5개 |
| 13:00 | 평일 | 새 학습 | 오늘의 단어 + 문법 1가지 |
| 15:00 | 평일 | 복습 | 점심에 배운 것 퀴즈 |
| 21:00 | 매일 | 일기 쓰기 | 오늘 하루를 스페인어로 |

**학습 타입 6가지**:
1. `greeting` — 인사말: 시간대별 자연스러운 인사
2. `situational` — 상황 단어: 맥락(운전, 직장, 식사 등)에 맞는 단어
3. `new_learning` — 새 학습: 20단계 커리큘럼 기반 단어 + 문법
4. `review` — 복습: 이전에 배운 것 퀴즈
5. `mistake_review` — 오답 복습: 틀린 것만 다시
6. `diary` — 일기 쓰기: 자유 작문 + AI 교정

## 4. AI 전략

**결정**: 로컬 LLM (Ollama) 사용  
**이유**: 비용 $0, RTX 5090 x2 (64GB VRAM)로 클라우드급 품질 가능

**현재 사용 모델**: llama3.3:latest (Q4, ~42GB VRAM)  
**검토 중 모델**: qwen2.5:7b-instruct (응답속도 향상 목적, 한국어·스페인어 다국어 강점)

**API**: Ollama 네이티브 `/api/chat` (stream 옵션 지원)  
**엔드포인트**: http://127.0.0.1:11434

**핵심 옵션** (반드시 매 호출에 포함):
```json
{ "num_ctx": 4096, "num_gpu": 99, "num_predict": 600 }
```

**AI 응답 규칙** (시스템 프롬프트에 강제):
- 설명·해석은 한국어로만
- 중국어 사용 절대 금지 (llama3.3이 다국어 혼용 경향 있음)
- 교정 형식: ⚠️ 내 답 → 정답 → 설명

## 7. 스페인어 커리큘럼

**20단계** — `backend/curriculum.py`에 전체 정의됨

| 레벨 | 주제 |
|------|------|
| 1 | 기초 인사 & 자기소개 (ser 동사) |
| 2 | 숫자 & 기본 동사 (-ar/-er/-ir) |
| 3 | 명사 & 관사 |
| 4 | 형용사 & 성수일치 |
| 5 | 소유형용사 |
| 6 | 의문문 & 부정문 |
| 7 | 불규칙 동사 핵심 |
| 8 | 전치사 & 목적격 a |
| 9 | 목적어 대명사 |
| 10 | 과거형 기초 (규칙) |
| 11~20 | 고급 문법 → 자유 표현 |

수업 형식 (get_level_prompt):
- 동사 1개 + 예문 2개
- 단어 1개 + 예문 2개
- 문법 포인트 1개
- 문제 5개 + 체크 3개
