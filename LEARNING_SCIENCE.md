## 1. 3대 핵심 원칙

### ① Retrieval Practice (인출 연습)
> "꺼내는 것이 다시 읽는 것보다 훨씬 강력"

- **대표 연구**: Henry L. Roediger III, Jeffrey D. Karpicke (2006)
- **결과**: 반복 읽기보다 회상 테스트가 장기 기억에서 압도적으로 우수
- **핵심**: 기억은 저장이 아니라 **"꺼낼 때 강화"**됨

### ② Spacing Effect (간격 효과)
> "몰아서보다 나눠서"

- **대표 연구**: Hermann Ebbinghaus (망각곡선), John Dunlosky (2013 메타연구)
- **핵심**: 망각 직전에 다시 보면 가장 효율적

### ③ Elaboration (정교화)
> "이해 + 연결 + 설명"

- **관련 효과**: Self-explanation effect
- **연구자**: Michelene T. H. Chi
- **핵심**: 단순 암기 < 의미 연결

## 2. 최적 간격 공식

> **"시험(또는 사용할 시점)까지 남은 시간의 10~20% 간격"**
> — Nicholas J. Cepeda et al. (2008)

| 목표 | 최적 복습 간격 |
|------|--------------|
| 1일 후 기억 | 몇 시간 간격 |
| 1주일 유지 | 하루 간격 |
| 1개월 유지 | 3~5일 간격 |

→ **고정된 간격이 아니라 목표 기간에 따라 달라짐**

---

## 3. 검증된 최적 학습 구조 (SRS)

```
학습 직후    → 이해 (Encoding)
몇 분~몇 시간 → 1차 인출
하루         → 2차 인출
3~7일        → 적용 + 문제풀이
1~4주        → 재인출
```

이 구조 = **Spaced Repetition System (SRS)** — Anki가 대표 구현체

## 4. 핵심 메커니즘

### Desirable Difficulty
> "약간 힘들게 떠올릴 때 기억이 강화됨"
- 연구자: Robert A. Bjork

### Forgetting + Retrieval
> "조금 잊은 상태에서 꺼내야 효과 최대"

### Storage vs Retrieval Strength
- 기억에는 두 가지 강도가 있음: **저장 강도(storage)** + **인출 강도(retrieval)**
- 인출 연습이 둘 다 동시에 강화

## 5. PicoPico 현재 구조 vs 과학적 최적화

### 현재 (MVP)
```
고정 시간 알림 (6시/9시/21시) → 해당 학습 타입 ChatScreen 진입
```
→ Spacing Effect O, Retrieval Practice O, **SRS X**

### 목표 (Phase 2)
```
새 학습 완료 → 3시간 후 복습 알림
            → 다음날 복습 알림
            → 3일 후 오답 복습 알림
            → 1주 후 재인출 알림
```
→ **개인화된 망각곡선 기반 알림** 필요
→ DB 설계 선행 필요 (학습 이력 + 다음 복습 시각 저장)
