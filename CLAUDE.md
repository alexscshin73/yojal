# Yojal — 개인맞춤형 스페인어·영어 학습 앱

## 프로젝트 개요

**Yojal** (요할) — 한국어 "조잘조잘/재잘재잘"에서 온 이름.
슬로건: *Habla más, aprende mejor.* (더 많이 말하고, 더 잘 배워라)

마스코트: 빨간 앵무새 (말하는 것을 즐기는 캐릭터)

## 개발 원칙 (Harness Engineering)

- 기능은 한 번에 하나씩 완성한 뒤 다음으로 넘어간다
- 새 세션 시작 시 반드시 이 파일을 먼저 읽는다
- 기능 완성 후 자동 검증(테스트)을 돌린다
- 규칙과 도구는 꼭 필요한 것만 사용한다

## 타겟 사용자

한국어 모국어 화자, 스페인어 또는 영어를 배우고 싶은 사람

## 핵심 기능 (우선순위 순)

1. **일상 대화 연습** — AI와 스페인어/영어로 대화 (텍스트)
2. **문법 교정** — 틀린 문장을 부드럽게 교정, 설명
3. **단어 퀴즈** — 사용자 레벨에 맞는 단어 반복 학습
4. **발음 교정** — 음성 입력 기반 피드백 (후순위)

## 기술 스택

- **Frontend**: Next.js (웹앱, Vercel 배포)
- **AI**: Anthropic Claude API (claude-sonnet-4-6)
- **언어**: TypeScript
- **스타일**: Tailwind CSS
