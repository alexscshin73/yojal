# SCREENS — 화면 설계 초안

> 상태: ✅ 승인 완료 (2026-04-23)
> 다음 단계: 디자인 (각 화면 상세 UI 설계)

---

## 전체 탐색 구조

```
처음 설치
    ↓
[온보딩] (처음 한 번만)
  1. 환영 화면
  2. 레벨 선택
  3. 루틴 설정
  4. 알림 권한
    ↓
[메인 앱]
    ↓
하단 탭: 홈 / 기록 / 설정

푸시 알림 → 채팅 화면 (직접 이동)
홈 화면 → 채팅 화면
```

---

## 화면 목록

### S-01. 온보딩 — 환영 화면
처음 설치 시 한 번만 표시

```
🦜 큰 앵무새 마스코트
"Hola! 저는 요할이에요"
"매일 스페인어로 조잘거릴게요"
[시작하기]
```

### S-02. 온보딩 — 레벨 선택
```
완전 초급  (스페인어 처음)
초급       (인사말 정도 앎)
중급       (간단한 대화 가능)
```

### S-03. 온보딩 — 루틴 설정
```
요일 × 시간 × 학습타입 설정
기본 루틴 제안 → 수정 가능
```

### S-04. 온보딩 — 알림 권한
```
"요할이 말 걸 수 있게 허락해줘요"
[허락하기]
```

---

### S-05. 홈 화면 (매일 보는 화면)
```
상단: 🔥 12일 연속 학습 | 오늘 배운 단어 3개

중간: 오늘 스케줄 타임라인
  ✅ 06:00 인사말 완료
  ▶️ 13:00 새 단어 · 30분 후
  ⏸  17:30 오답 복습
  ⏸  21:00 일기 쓰기

하단: [지금 바로 연습하기]
```

---

### S-06. 채팅/학습 화면 (핵심 화면)
```
상단: [← 뒤로]  학습 타입 표시  [?힌트]

메시지 영역:
  🦜 AI 메시지 (스페인어)
     학습 타입 뱃지 표시
  → 내 답장
  🦜 교정 + 다음 메시지

하단: [스페인어로 답장...]  [전송]
```

**학습 타입별 AI 행동**:
| 타입 | AI 행동 |
|------|---------|
| 인사말 | 시간대별 자연스러운 인사 건넴 |
| 상황 단어 | 오늘 주제 단어 5개 제시 |
| 새 학습 | 단어 + 문법 1가지 설명 |
| 복습 | 이전 내용 퀴즈 |
| 오답 복습 | 틀린 것만 다시 |
| 일기 쓰기 | 자유 작문 유도 + 교정 |

---

### S-07. 루틴 설정
```
요일 탭: [월] [화] [수] [목] [금] [토] [일]

선택된 요일의 루틴 목록:
  06:00  인사말     [편집] [삭제]
  08:00  상황 단어  [편집] [삭제]
  13:00  새 학습    [편집] [삭제]
  21:00  일기 쓰기  [편집] [삭제]

  [+ 시간 추가]
[저장]
```

---

### S-08. 학습 기록
```
탭: [단어장] [오답노트] [통계]

단어장:
  이번 주 배운 단어 · 23개
  conducir    운전하다  ✅
  el semáforo 신호등    ✅
  aparcar     주차하다  ❌ (2번 틀림)

오답노트:
  aparcar — 주차하다 (3번 틀림)
  [다시 연습하기]

통계:
  주간 학습일, 단어 수, 스트릭 캘린더
```
